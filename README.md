Creatorsphere
The world's first life operating system — connecting content creation, financial growth, skill sharing, recovery support, disability navigation, and community funding in one interconnected ecosystem.
Free forever. Built in bateau Australia.
What is Creatorsphere?
Every other platform serves the top 1% of creators. Creatorsphere serves everyone else.
It's a single app where a creator can plan content, a single parent can budget on $400 a week, someone in recovery can privately track their progress, an NDIS participant can navigate their plan in plain language, and a community can fund each other's dreams — all connected, all in one place.
No separate apps. No monthly fees for basic features. No being left behind because you can't afford the "pro" version.

Tech Stack
Frontend: React + TypeScript + Tailwind CSS
Backend: Node.js + Express + TypeScript
Database: PostgreSQL via Drizzle ORM (Neon serverless)
Auth: Passport.js (local + Google OAuth)
File Storage: Multer (100MB upload limit)
Sessions: PostgreSQL session store
Routing: SPA with 28 registered routes

Quick Start
git clone https://github.com/YOUR_USERNAME/creatorsphere.git
cd creatorsphere
npm install
cp .env.example .env
# Edit .env with your DATABASE_URL and SESSION_SECRET
npm run db:push
npm run dev
Deploy to Render.com (Free)
Fork this repository
Create a free account on render.com
New  Web Service connect your GitHub
Build command npm install && npm run build
Start command npm start
Add environment variables (DATABASE_URL, SESSION_SECRET)
